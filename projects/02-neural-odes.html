Neural ODEs para dinámica de sistemas

Una red neuronal aprende el campo vectorial dy/dt = f(y, t) de un sistema dinámico (péndulo amortiguado, Lotka-Volterra) directamente desde trayectorias observadas, usando integradores numéricos diferenciables.

Código en GitHub → Demo (pendiente)
// Resumen

¿De qué se trata este proyecto?

Las redes neuronales clásicas aprenden funciones directas x → y. Las Neural ODEs (Chen et al., NeurIPS 2018) aprenden la derivada dy/dt = fθ(y, t) y la integran con un solver numérico, haciendo el proceso diferenciable mediante el método del adjoint.

Este proyecto aprovecha la ventaja natural del perfil físico-matemático: si entendés EDOs bien, la arquitectura es intuitiva — una MLP que aproxima un campo vectorial, integrada con Runge-Kutta dentro del grafo de autograd.

Meta: entrenar con trayectorias cortas (ej. 5 segundos) y verificar que la red extrapola correctamente a horizontes más largos, algo que una red clásica no puede hacer porque no aprende la dinámica subyacente.

Conceptos clave

Neural ODE:

dy/dt = fθ(y, t)

Integración y loss:

y(ti) = y₀ + ∫ fθ dt
L = Σᵢ ‖y(tᵢ) − yᵢobs‖²

Sistema de referencia (péndulo amortiguado):

θ̈ + γθ̇ + ω₀² sin θ = 0
// Skills demostrados

Qué habilidades certifica

PyTorch
Definir módulos nn.Module, training loop con Adam, uso correcto de autograd.
Integradores diferenciables
Uso de torchdiffeq.odeint con métodos Dormand-Prince y tolerancias adaptativas.
ML matemático
Regresión de campo vectorial en ℝ², no solo clasificación — el objetivo es estructura continua.
Generación de datos
Integrar el sistema real con scipy.integrate.odeint y añadir ruido gaussiano controlado.
Experimentación
Ablation: tamaño de red, profundidad, ruido, longitud de trayectoria de training.
Visualización
GIF animado comparando trayectoria real vs. predicha y campo vectorial aprendido en el plano de fase.
Validación física
Chequear conservación aproximada de energía y verificar extrapolación fuera del rango de entrenamiento.
PyTorch torchdiffeq NumPy SciPy Matplotlib Autograd EDOs Jupyter
