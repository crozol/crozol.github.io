Neural ODEs for dynamical systems

A neural network learns the vector field dy/dt = f(y, t) of a dynamical system (damped pendulum, Lotka-Volterra) directly from observed trajectories, using differentiable numerical integrators.

Code on GitHub → Demo (pending)
// Overview

About this project

Classic neural networks learn direct functions x → y. Neural ODEs (Chen et al., NeurIPS 2018) learn the derivative dy/dt = fθ(y, t) and integrate it with a numerical solver, making the process differentiable via the adjoint method.

This project leverages a natural advantage of the physics-mathematics background: if you understand ODEs well, the architecture is intuitive — an MLP approximating a vector field, integrated with Runge-Kutta inside the autograd graph.

Goal: train on short trajectories (e.g. 5 seconds) and verify the network extrapolates correctly to longer horizons — something a classical NN can't do because it doesn't learn the underlying dynamics.

Key concepts

Neural ODE:

dy/dt = fθ(y, t)

Integration and loss:

y(ti) = y₀ + ∫ fθ dt
L = Σᵢ ‖y(tᵢ) − yᵢobs‖²

Reference system (damped pendulum):

θ̈ + γθ̇ + ω₀² sin θ = 0
// Skills demonstrated

What skills it certifies

PyTorch
Define nn.Modules, training loop with Adam, correct use of autograd.
Differentiable integrators
Use of torchdiffeq.odeint with Dormand-Prince methods and adaptive tolerances.
Mathematical ML
Vector field regression in ℝ², not just classification — the target is continuous structure.
Data generation
Integrate the real system with scipy.integrate.odeint and add controlled Gaussian noise.
Experimentation
Ablation: network size, depth, noise, training trajectory length.
Visualization
Animated GIF comparing real vs. predicted trajectory and learned vector field in the phase plane.
Physical validation
Check approximate energy conservation and verify extrapolation beyond the training range.

Approximate energy conservation in the pendulum (drift < 5%).
