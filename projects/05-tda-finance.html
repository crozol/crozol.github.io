05 / 05 Topología aplicada · Finanzas Planeado

TDA en series financieras
Detección de cambios de régimen

Usar homología persistente sobre ventanas deslizantes de retornos multi-activo para detectar transiciones entre régimen normal y crisis de mercado (2008, 2011, 2015, 2020). Los descriptores topológicos alimentan un clasificador supervisado.

Código en GitHub → Demo (pendiente)
// Resumen

¿De qué se trata este proyecto?

Los métodos clásicos para detectar cambios de régimen en finanzas (volatilidad rolling, correlaciones móviles, HMMs) miran momentos estadísticos. El Topological Data Analysis propone un enfoque complementario: mirar la forma de la nube de retornos en ℝd mediante homología persistente, y extraer invariantes robustos ante reparametrización.

Idea: en régimen normal los retornos forman una nube "redonda" de baja correlación; durante una crisis, la nube colapsa hacia una dirección (co-movimiento sistémico) y su topología cambia. Las barras de persistencia detectan ese cambio antes que métricas agregadas tradicionales.

Este proyecto usa tu formación en topología algebraica — una herramienta que ningún MLE junior va a tener — para un problema financiero con impacto real.

Pipeline matemático

Retornos log:

rt = log(pt / pt−1) ∈ ℝd

Ventana deslizante:

Wt = {rs : t − h ≤ s ≤ t} ⊂ ℝd

Homología persistente Vietoris-Rips:

PD0(Wt), PD1(Wt)

Features:

φ(Wt) = [entropía, landscape, n. componentes, persistencia máxima]
// Skills demostrados

Qué habilidades certifica

Topología algebraica aplicada
Cálculo de homología persistente Vietoris-Rips en dimensiones 0 y 1 sobre nubes de puntos.
Feature engineering exótico
Convertir diagramas de persistencia en vectores (persistence landscapes, entropía, images) para ML clásico.
Series temporales
Ventanas deslizantes, alineamiento temporal, manejo de ticks faltantes y días festivos.
Finanzas cuantitativas
Retornos log, volatilidad realizada, correlaciones rolling como baseline de comparación.
ML clásico
Random Forest / SVM sobre features topológicos con validación cruzada temporal (walk-forward).
Validación de dominio
Confirmar que los picos del score coinciden con eventos macro reales (Lehman 2008, COVID 2020).
Storytelling visual
Gráficas antes / durante / después de crisis con diagramas de persistencia anotados.
Python Giotto-TDA Ripser NumPy Pandas scikit-learn yfinance Matplotlib Persistence Homology

// Roadmap

Progreso del proyecto

  • Descargar precios diarios (SP500, NASDAQ, sector ETFs) via yfinance
  • Preprocessing: retornos log, normalización, manejo de gaps
  • Ventanas deslizantes → nubes de puntos en ℝd
  • Calcular homología persistente (dim 0 y 1) con Giotto-TDA
  • Convertir diagramas en persistence landscapes y entropía
  • Entrenar Random Forest con validación temporal walk-forward
  • Graficar score temporal con anotaciones de crisis reales
  • README con narrativa + notebook interactivo
